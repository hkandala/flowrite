# keyboard shortcuts

complete reference of all keyboard shortcuts in flowrite.

## general

| shortcut | action |
|----------|--------|
| `⌘N` | new file |
| `⌘O` | open file |
| `⌘S` | save |
| `⌘⇧S` | save all |
| `⌘W` | close tab |
| `⌘P` | command palette |
| `Ctrl+Tab` | next tab |
| `Ctrl+⇧+Tab` | previous tab |
| `⌘1`–`⌘9` | focus editor group |

## panels & layout

| shortcut | action |
|----------|--------|
| `⌘⇧E` | toggle file tree |
| `⌘⇧L` | toggle AI chat |
| `⌘⇧=` | toggle zen mode |
| `⌘⇧-` | toggle full width |

## AI & chat

| shortcut | action |
|----------|--------|
| `⌘L` | ask AI / add file to chat |
| `Enter` | send message |
| `Shift+Enter` | new line in chat |

## text formatting

| shortcut | action |
|----------|--------|
| `⌘B` | bold |
| `⌘I` | italic |
| `⌘U` | underline |
| `⌘⇧M` | strikethrough |
| `⌘E` | inline code |
| `⌘K` | link |
| `⌘D` | comment |

## blocks

| shortcut | action |
|----------|--------|
| `⌘⌥1`–`⌘⌥6` | heading 1–6 |
| `⌘⌥8` | code block |
| `⌘⌥X` | toggle todo check |
| `⌘Enter` | insert block after |
| `⌘⇧Enter` | insert block before |

## editor navigation

| shortcut | action |
|----------|--------|
| `⌘⌥J` | next block |
| `⌘⌥K` | previous block |
| `⌘⌥L` | next section |
| `⌘⌥H` | previous section |
